# 🎓 Student Record System (C)

A menu-driven student record management system implemented in C using structures and arrays.

---

## 🚀 Features
- Add student records
- Display all students
- Search student by roll number

---

## 🛠️ Technologies Used
- C Programming
- Structures
- Arrays

---

## ▶️ How to Run
1. Compile the code using a C compiler (GCC)
2. Run the executable file
3. Follow menu options

---

- Understanding of structures in C
- Hands-on with menu-driven programs
- Improved problem-solving skills
